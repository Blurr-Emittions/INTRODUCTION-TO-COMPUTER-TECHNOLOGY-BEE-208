# BEE 208: Control Statements in C++ — Lab Activity Set 2

## Course Details
* **Institution:** Accra Technical University (ATU)
* **Faculty:** Faculty of Engineering
* **Department:** Department of Electrical/Electronic Engineering
* **Course Code:** BEE 208

## Description
This folder contains the complete engineering documentation and logic design deliverables for the 20 practical questions in Lab Activity Set 2. The exercises focus on implementing fundamental programmatic logic—ranging from conditional logic (if...else if...else) to loop structures (for, while, do-while) and control break/continue statements—tailored to electrical and electronic engineering contexts.

## Student Deliverables Included
Per the marking guide of Accra Technical University, each problem contains:
1. **Algorithm:** A sequential, step-by-step logical design.
2. **Pseudocode:** Text-based structural language mapping out the C++ architecture.
3. **Flowchart:** Graphical system representations showing the exact program execution paths.
4. **Test Results:** Output evidence and test cases validating the accuracy of the logic.
